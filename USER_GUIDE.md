# GiftDrop User Guide 🎁

## Step 1 — Install Freighter Wallet
1. Go to freighter.app
2. Install browser extension
3. Create a new wallet
4. Switch network to Testnet

## Step 2 — Get Free Testnet XLM
1. Go to https://friendbot.stellar.org
2. Enter your wallet address → Submit → You get 10,000 free XLM!

## Step 3 — Create a Gift Drop
1. Go to https://giftdrop-five.vercel.app
2. Click "Create a Gift Drop"
3. Enter recipient wallet, target amount, reveal date
4. Click Create → sign with Freighter

## Step 4 — Share With Friends
Copy the drop link and send via WhatsApp/Telegram.

## Step 5 — Contribute
1. Open drop link → Enter XLM amount → Click "Lock Funds"
2. Sign with Freighter → check Stellar Explorer for confirmation

## Step 6 — Reveal
On reveal date, funds automatically release to recipient!

## FAQ
**Q: Target not reached?** → Everyone gets auto-refunded.  
**Q: Is my money safe?** → Yes, locked in Soroban smart contract.  
**Q: Which wallet?** → Freighter browser extension.
